# 7. 스프링 삼각형과 설정 정보
스프링을 제대로 이해하기 위해서는 POJO(Plain Old Java Object)를 기반으로, 스프링 삼각형이라고 불리우는 IoC/DI, AOP, PSA라고 하는 스프링의 3대 프로그래밍 모델에 대한 이해가 필요하다. <br> **스프링 프레임워크는 스프링 삼각형의 조합으로 이해할 수 있다.**
